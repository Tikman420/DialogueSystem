I originally made this tool for unity but I wanted to move to a custom engine + I wanted to try ImGui out.
right now it is ***only*** displaying the dialogue located at Game/Test.txt

**Sources (REQUIRED to build):**
================================
* [SFML (v3.0.2)](https://www.sfml-dev.org/download/)
* [Dear ImGui (v1.91.1)](https://github.com/ocornut/imgui/releases/tag/v1.91.1)
* [ImGui-SFML](https://github.com/SFML/imgui-sfml/tree/master)  

(Follow the manual install instructions on ImGui-SFML and you'll probably be fine)
  (also add "opengl32.lib" to Linker > Input > Additional Dependencies)
